# homebrew-tap

개인 Homebrew 저장소입니다.

## 설치 방법

```bash
brew tap ljdongz/tap
```

## Formula

### codegate

텔레그램에서 Claude Code Channel 세션을 원격으로 관리하는 CLI 도구입니다.

```bash
brew install ljdongz/tap/codegate
```

자세한 내용은 [codegate 저장소](https://github.com/ljdongz/codegate)를 참고하세요.

## Cask

### Scrab

macOS 화면 캡처 도구입니다.

```bash
brew install --cask ljdongz/tap/scrab
```

자세한 내용은 [Scrab 저장소](https://github.com/ljdongz/Scrab)를 참고하세요.

## 삭제

```bash
brew untap ljdongz/tap
```
